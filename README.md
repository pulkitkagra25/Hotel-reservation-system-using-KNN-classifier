# 🏨 Hotel Reservation Prediction using KNN

## 📌 Description

This project uses Machine Learning to predict whether a hotel reservation will be **canceled or not**. It is built using the K-Nearest Neighbors (KNN) algorithm, which classifies bookings based on similarity with past data.

The model analyzes customer and booking details to make accurate predictions.

---

## 🚀 Features

* Predicts booking status (Canceled / Not Canceled)
* Data preprocessing and cleaning
* KNN algorithm implementation
* Model evaluation with accuracy metrics
* Easy-to-understand and structured code

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

---

## 📂 Dataset

The dataset includes features like:

* Booking ID
* Number of adults and children
* Meal plan type
* Room type reserved
* Lead time
* Arrival date
* Market segment type
* Average price per room
* Booking status

---



---

## ▶️ Usage

Run the main file:

```bash
python hotel_reservation.ipynb
```

The program will:

* Load and preprocess the dataset
* Train the KNN model
* Predict reservation status
* Display results and accuracy
